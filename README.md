# Place-Pick-App

## 폴더 구조

- assets: 이미지,폰트 폴더

  - icons: 아이콘
  - images: 이미지

- components: 컴포넌트 폴더

  - customComponents: react native 컴포넌트 커스텀
  - home: home screen 관련 컴포넌트

- constants: 상수 폴더

- hooks: hook 폴더

- lib: 라이브러리 관련 폴더

- navigation: navigator 폴더

- screens: 스크린 모음 폴더

- store: 상태관리 폴더

- types: type 폴더

  - navigation: navigation 관련 타입

- utils: 함수 및 유틸리티 코드 폴더
